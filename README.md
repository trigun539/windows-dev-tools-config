# Easy configuration for development tools for Windows

## Dependencies

- Vim
- Git
- NodeJS
- Mono
- Visual Studio
- Java
- Maven
- Python
- Ctags

## Tools

- Vim

## How to use

- Clone the repo
- Replace .vimrc
- Add vimfiles/bundle folder to vimfiles
- copy vim80/ files to vim install directory
- Clone ultisnips repo to the vimfiles directory
